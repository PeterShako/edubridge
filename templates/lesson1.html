Lesson 1: Introduction to HTML

What is HTML?
HTML is actually shorthand for HyperText Markup Language. It is the language of web pages that tells a browser how to display certain elements, such as text and images, through the use of codes and symbols.

Watch the Video: Introduction to HTML

HTML is the standard when it comes to creating web pages. The World Wide Web Consortium, or W3C, recommends it. Being such, most browsers implement HTML to help display web pages more or less uniformly.

HTML is the brainchild of Tim Berners-Lee. In 1990, Berners-Lee needed something that would help scientists from different colleges and universities access documents and research from other scientists. This problem led to Berners-Lee inventing the World Wide Web, the hypertext transfer protocol (HTTP), and HTML.

Why learn HTML?
These days, the proliferation of content management systems (CMS) like WordPress, as well as HTML editors, can help you create a web page without knowing HTML. However, even with these tools, knowing how HTML works (plus knowing just the basic tags) can go a long way.

  • HTML makes it easier to correct design flaws (e.g., fixing image alignment).
  • It’s simple and easy to learn, making it an essential skill for any aspiring web developer.

What you need to get started:
Learning HTML doesn’t require much. You can use a basic text editor like Notepad or a more advanced tool like an HTML editor or WYSIWYG editor (What You See Is What You Get). However, when learning, we recommend hardcoding HTML using a text editor to understand the structure and syntax better.

To get started with HTML, focus on these two things:

  • The basic tags and elements.
  • The structure of an HTML page.
