HTML Tables Are Very Flexible

As you can see, HTML tables are very flexible. Aside from allowing you to have exactly the cells, rows and columns that you need, you can also put in any data you want. For example, you could have words, paragraphs, bulleted lists and even another table within a table.

Text Boxes and Other User Input "Thingamajigs"

HTML forms provide a great way to capture data from your website. A form in an HTML page can be used to submit data to a server for further processing.

Each HTML form is introduced by the <form> tag, and is made up of different input elements. The <form> tag has several attributes that would tell the browser what to do with the information that is entered in the different fields. These attributes are:

  • action
  • method

The action attribute tells your browser the location of the CGI script that you are going to use to process the form. On the other hand, the method attribute has two possible values, method="get" or method="post", both of which specify a method of submitting data to the script you have specified in the action attribute.
